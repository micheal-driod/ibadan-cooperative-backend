# IBARFFS COOPs Management System

A full-stack cooperative savings and loan management system built for FAAN workers in Ibadan.

## 🚀 Overview

This system enables cooperative members to:
- Log in using staff number
- Apply for loans with guarantor verification
- Track savings, shares, and loan balances

Admins can:
- Manage members
- Process loan applications
- Upload monthly financial records
- Monitor system activity

---

## 🛠 Tech Stack

- Backend: Node.js (Express) + Prisma ORM
- Database: PostgreSQL (Supabase)
- Admin Panel: HTML/CSS/JS (Deployed on Vercel)
- Mobile App: Flutter (Android + Web)
- Hosting:
  - Backend → Railway
  - Database → Supabase
  - Admin → Vercel
  - Mobile Web → Vercel
